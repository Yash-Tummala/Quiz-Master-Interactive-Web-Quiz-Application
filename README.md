## 📖 About

Quiz Master is a fully browser-based quiz app that challenges your knowledge across four hand-picked topics — **Python Programming**, **Science**, **AI Technology**, and **Cricket**. Every round feels fresh because questions are shuffled randomly, so you'll never get the same order twice.

This project was built from scratch as part of my frontend learning journey to master the fundamentals of web development — DOM manipulation, event-driven programming, and managing UI state — all without relying on any external library or framework.

---

## ✨ Features

- 🎯 **4 Topics** — Python, Science, AI Technology, Cricket
- 🔀 **Random Shuffle** — questions reordered on every attempt
- ✅ **Instant Feedback** — correct/wrong answer highlighted after each pick
- 📊 **Live Score & Progress Bar** — tracks your score question by question
- 🏆 **Results Screen** — accuracy %, correct vs wrong count, full per-question review
- 🔁 **Retry or Switch** — replay the same topic or pick a new one
- 🌙 **Dark Theme UI** — clean, minimal dark interface with smooth transitions
- 📱 **Responsive** — works on desktop and mobile browsers
